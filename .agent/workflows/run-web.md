# Run Flutter Web App on Chrome (Port 8080)

Flutter 웹 앱을 크롬에서 8080 포트로 실행합니다.

## Steps

// turbo
1. Run the app on Chrome with fixed port:
```bash
flutter run -d chrome --web-port 8080
```

## 참고

- 앱은 `http://localhost:8080`에서 실행됩니다
- Hot reload는 `r` 키로 가능합니다
- 종료는 `q` 키를 누르세요
